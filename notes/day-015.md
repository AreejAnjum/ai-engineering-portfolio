# Day 15 - JavaScript Foundation Map

## Goal

Today I started Week 3: JavaScript Fundamentals. The goal was to plan how JavaScript will upgrade my Personal Learning Dashboard from a static page into an interactive dashboard.

## What I learned today

1. JavaScript adds behavior and interaction to a web page.
2. HTML gives structure, CSS gives style, and JavaScript gives logic.
3. Arrays can store lists of items, such as learning tasks.
4. Objects can store structured data, such as task title, category, and status.
5. The DOM lets JavaScript read and update HTML elements on the page.

## Key concepts

| ---------- | ----------------------------------------------- |
| JavaScript | Adds logic and interaction to a web page        |
| Variable   | Stores one value                                |
| Array      | Stores a list of values                         |

## What was difficult?

Understanding how JavaScript connects to HTML.

## How I solved it

I learned the simple frontend model:

```text
HTML = structure
CSS = design
JavaScript = behavior
```

This means my dashboard can later show tasks, filter tasks, and update content without changing the HTML manually every time.

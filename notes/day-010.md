# Day 10 - Forms

## Goal

Today I added a learning task form to my Personal Learning Dashboard. The goal was to create usable and accessible form inputs for adding a task title, category, and notes.

## What I learned today

1. HTML forms collect user input.
2. The `<form>` tag groups input fields together.
3. The `<label>` tag makes inputs easier to understand and more accessible.
4. Inputs should have matching `id` and `for` values.
5. CSS can make forms cleaner, easier to use, and more professional.

## Key concepts

| Concept       | Simple meaning                            |
| ------------- | ----------------------------------------- |
| Form          | A section used to collect user input      |
| Input         | A field where the user types data         |
| Label         | Text connected to an input field          |
| Select        | A dropdown field                          |
| Textarea      | A larger text input area                  |
| Focus state   | Visual style when an input is selected    |
| Accessibility | Making the UI easier for all users to use |

## Key concepts

| Concept         | Simple meaning                                    |
| --------------- | ------------------------------------------------- |
| JavaScript      | Adds logic and behavior to a web page             |
| Script file     | A `.js` file connected to HTML                    |
| Array           | A list of values                                  |
| Object          | Data stored with key-value pairs                  |
| DOM             | The page structure JavaScript can read and update |
| `querySelector` | Finds an HTML element                             |
| `createElement` | Creates a new HTML element                        |
| `appendChild`   | Adds an element to the page                       |

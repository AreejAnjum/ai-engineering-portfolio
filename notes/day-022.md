## What was difficult?

Understanding why TypeScript matters before using React or Next.js.

## How I solved it

I learned that TypeScript is useful because it makes data rules clear.

```text
Without TypeScript:
A task can accidentally have missing or wrong fields.

With TypeScript-style thinking:
A task should always follow a clear structure.
```

Example task structure:

```text
title: text
category: text
status: allowed status
notes: text
```

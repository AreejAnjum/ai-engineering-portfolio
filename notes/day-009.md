# Day 9 - CSS Layout

## Goal

Today I improved the Personal Learning Dashboard by adding CSS layout, spacing, typography, cards, and responsive design.

## What I learned today

1. CSS controls how an HTML page looks.
2. `font-family`, `color`, and `background` improve visual style.
3. `padding`, `margin`, and `gap` control spacing.
4. `border-radius`, `border`, and `box-shadow` can make sections look like cards.
5. Media queries help the page look good on both mobile and desktop.

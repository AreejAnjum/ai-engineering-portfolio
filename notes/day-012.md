# Day 12 - Deployment Preview

## Goal

Today I deployed my Personal Learning Dashboard online. The goal was to create a live URL so the project can be opened and reviewed in a browser.

## What I learned today

1. Deployment means publishing a project online.
2. A live URL makes a portfolio project easier to share.
3. Before deployment, I should test the project locally.
4. Vercel can deploy static HTML/CSS projects from GitHub.
5. A project is stronger when the README includes a live demo link.

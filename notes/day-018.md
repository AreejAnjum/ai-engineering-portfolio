# Day 18 - JavaScript Integration Day

## Goal

Today I connected the dashboard form with JavaScript task rendering. The goal was to create an end-to-end flow where a user can submit the form and see the new task appear on the page.

## What I learned today

1. JavaScript can listen for form submissions using `addEventListener`.
2. `event.preventDefault()` stops the page from refreshing after form submit.
3. Input values can be read using `.value`.
4. New task objects can be added to an array using `push()`.
5. Re-rendering updates the page after new data is added.

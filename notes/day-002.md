# Day 2 - Terminal Basics

## Goal

Today I learned how to use the terminal to move inside my project, create files, create folders, read files, and delete files safely.

## What I learned today

1. `pwd` shows my current folder location.
2. `ls` shows files and folders inside the current folder.
3. `cd` moves from one folder to another.
4. `cd ..` goes one folder back.
5. `mkdir -p` creates folders safely.
6. `touch` creates an empty file.
7. `echo` can write text into a file.
8. `cat` shows the content of a file.
9. `rm` deletes a file, so I must use it carefully.

## Commands I practiced

```bash
cd ~/Developer/ai-engineering-portfolio
pwd
ls
mkdir -p docs/evidence/week-01/day-02
touch notes/day-002.md
cd notes
pwd
ls
cd ..
touch terminal-practice.txt
echo "Today I practiced terminal basics." > terminal-practice.txt
cat terminal-practice.txt
rm terminal-practice.txt
```

## Important command meanings

| Command    | Meaning                  |
| ---------- | ------------------------ |
| `pwd`      | Shows where I am         |
| `ls`       | Lists files and folders  |
| `cd`       | Changes folder           |
| `cd ..`    | Goes one folder back     |
| `mkdir -p` | Creates folders safely   |
| `touch`    | Creates a new empty file |
| `echo`     | Prints or writes text    |
| `cat`      | Shows file content       |
| `rm`       | Deletes a file           |

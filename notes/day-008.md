# Day 8 - HTML Skeleton

## Goal

Today I started Project 1: Personal Learning Dashboard. The goal was to create a basic semantic HTML page that opens in the browser.

## What I learned today

1. HTML gives structure to a web page.
2. `<!DOCTYPE html>` tells the browser this is an HTML document.
3. `<head>` contains page information like title and metadata.
4. `<body>` contains the visible page content.
5. Semantic tags like `<header>`, `<main>`, `<section>`, `<article>`, and `<footer>` make the page easier to understand.

## Key concepts

| Concept       | Simple meaning            |
| ------------- | ------------------------- |
| HTML          | Structure of a web page   |
| Semantic HTML | Meaningful HTML tags      |
| `header`      | Top intro area of a page  |
| `main`        | Main content area         |
| `section`     | A grouped part of content |
| `article`     | Independent content block |
| `footer`      | Bottom area of a page     |
